# flyrix

A music App using Flutter that can provide lyrics of tracks using third party API services.

## Build Setup And Login Instructions

```bash
# install project
$ git clone https://github.com/CyberIkshit/flyrix.git

# open app directory
$ cd flyrix

# install dependencies
$ pub get

# build/run

```
